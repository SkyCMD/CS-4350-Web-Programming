CSS Measurement Units

  • %: Defines a measurement as a percentage.
  • cm: in centimeters
  • in: in inches
  • mm: in millimeters
  • px: in pixels
  • : in points. A point is 1/72nd of an inch
  • em: A relative measurement for font height. Each 'em' unit is 12 pt.
  • ex: Relative to a font's 'x' heigh
  • pc: measurement in picas - a pica = 12 points
  • vw: viewport width, stands for a percentage of browser's width
  • vh: viewport height, stands for a percentage of browser's height
